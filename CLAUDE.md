# Claude 작업 규칙

Claude는 이 프로젝트 작업을 시작하기 전에 반드시 [AGENTS.md](AGENTS.md)를 먼저 읽고 따른다.

이 파일은 Claude 전용 진입점이다. `AGENTS.md`는 규칙 인덱스이고, 실제 규칙의 단일 출처는 `AGENTS.md`에 연결된 `docs/` 문서들이다. 규칙을 수정할 때는 이 파일이 아니라 링크된 상세 문서를 먼저 갱신한다.

구현 전 설계 단계에서는 [실제 구현 계획](docs/implementation-plan.md)과 [초기 세로 슬라이스](docs/initial-vertical-slice.md)를 단일 출처로 둔다. 코드가 문서와 달라지면 [PR 체크리스트](docs/pr-checklist.md)에 따라 처리한다.
